🎓 Student Record Management Using Structure (C++)

📖 Overview

This project demonstrates the use of structures ("struct") in C++ to store and manage student and course information. A structure is used to group related data into a single unit, making data organization simple and efficient.

The program initializes a student record with predefined values and displays the stored information on the screen.

---

✨ Features

- Stores student information using a structure.
- Maintains course-related details along with student data.
- Uses compile-time initialization.
- Displays records using structure member access (".") operator.
- Beginner-friendly example of C++ structures.

---

🏗️ Structure Details

Structure: "student"

The structure contains the following data members:

Data Member| Description
"studentID"| Student identification number
"studentName"| Student name
"courseCode"| Course code
"courseName"| Course name
"courseMarks"| Marks obtained in the course
